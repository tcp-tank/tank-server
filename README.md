### `tank-protocol` commands 

| Command     | Description                       |
| :---------- | :-------------------------------- |
| LOG         | Login user. |
| REG         | Register new user. |
| QUIT        | Disconnect. |
| STAT        | Returns information of server, including address and port to which the client should connect. |
| ACCT        | User account information. |
| DELE        | Delete user. |
